# AGENTS

This directory contains the Elixir implementation of `specs-runner`.

The specification files are in the directory [specs/elixir](../../specs/elixir).

## Rules

- Add `@moduledoc false` to test modules under `test/`
- Use `mix check` to check code format, compilation and style/consistency (with Credo)
- Use `mix test` to test code changes as needed

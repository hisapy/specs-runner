# Security Policy

## Supported versions

Security fixes are applied to the latest code on `main`.

## Reporting a vulnerability

Please do **not** open public issues for security vulnerabilities.

Report privately to: security@hisapy.dev

Include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Affected commit, tag, or branch (if known)
- Suggested mitigation (if available)

We will acknowledge reports as quickly as possible and coordinate disclosure and fixes with reporters.
